# Lottery Picker

A client-side web app for generating statistically-informed Mega Millions and Powerball number picks.

## Features

- **5 pick strategies** — Hot Streak, Due Numbers, Balanced, Smart Random, True Random
- **Frequency-colored balls** — ball colors reflect historical draw frequency
- **Live countdown** to the next draw for each game
- **Save up to 20 pick sets** per game with localStorage
- **Auto-check results** against official NY Open Data draw history
- **Export picks as PNG** to share or save
- Dark mode by default with a light mode toggle

## Running locally

```bash
npm install
npm run dev
```

## Tech

React + Vite, custom CSS, no backend required.
